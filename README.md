## 6.暂定的主要功能
     (1)获取定位权限，获取当前位置，根据经纬度判断城市后，查询此时所在城市的天气；
     (2)可添加多个城市，最多可查询的城市数待定；
     (3)显示实时天气；
     (4)显示三天天气预报；
     (5)点击进入今日生活指数页面；
     (6)天气状况与生活指数添加图标显示；
     (7)“关于”页面显示软件版本，作者，隐藏彩蛋；
     (8)极光推送，本地发起系统级通知，主要是提醒查看天气。
     (9)其他功能待定
